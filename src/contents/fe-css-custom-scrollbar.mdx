---
title: CSS 自定义滚动条
category: ['Frontend']
slug: fe-css-custom-scrollbar
date: 2024-08-25
summary: CSS 自定义滚动条样式。
---


```css
::-webkit-scrollbar {
  width: 6px;
  height: 6px;
}

::-webkit-scrollbar-track {
  background: none;
}

::-webkit-scrollbar-thumb {
  background-color: #888;
  border-radius: 10px;
  border: none;
}

::-webkit-scrollbar-thumb:hover {
  background: #555;
}
```
