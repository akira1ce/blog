# 资源嵌套

将所有不常用的文件收纳到一个目录「非真实目录」嵌套下面。

```js
// .vscode/setting.json
{
  "explorer.fileNesting.enabled": true,
  "explorer.fileNesting.patterns": {
    "vite.config.ts": ".*rc, .cursor*,*.ini, *.sh, *.json, *.config.js, index.html, README.md, .git*"
  }
}
```
